# Cookies and Local Storage

This project demonstrates working with cookies and local storage in JavaScript.

## Tasks

### 0. Create basic cookie
Creating a basic HTML page with cookie management functionality using vanilla JavaScript.

## Requirements
- All files end with a new line
- Code uses the `.js` extension
- All code compiled/executed on Ubuntu 20.04 LTS using Node 20.x.x and npm 10.x.x
- `src/index.js` should stay empty - all JavaScript must be in HTML inside `<script>` tag

## Setup
```bash
npm install
node_modules/.bin/webpack-dev-server
```

Access the page at: http://localhost:8080/0-index.html
