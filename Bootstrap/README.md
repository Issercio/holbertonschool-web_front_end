# Bootstrap Project

This project demonstrates the use of Bootstrap 4.4.1 for responsive web design and CSS styling.

## Technologies Used
- Bootstrap 4.4.1 (CDN)
- HTML5
- CSS3

## Files
- `0-index.html` - Main page with Bootstrap styling
- `1-index.html` - Responsive blocks layout using Bootstrap grid system
- `2-index.html` - Responsive table of chemical elements with dark styling
- `3-index.html` - Cards with responsive layout and modal form
- `4-index.html` - Bob Dylan page with tabs and collapsible discography
- `5-index.html` - Show/Hide content based on screen size
- `6-index.html` - Pricing page with custom styled components
- `6-styles.css` - Custom stylesheet overriding Bootstrap default colors
